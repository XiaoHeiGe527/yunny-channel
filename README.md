# yunny-channel

#### 介绍
小黑哥的第一个练手项目

#### 软件架构
软件架构说明
Spring Boot +mybatis

rabbitmq、redis、@Async 多线程


#### 文件夹说明
job xxl-job 徐雪丽的定时任务 需要部署该服务

thirdpay  第三方服务对接  阿里与微信

oss  第三方文件上传服务 阿里

SMS 第三方短信服务 阿里 乐信
